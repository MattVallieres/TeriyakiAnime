# TeriyakiAnime

### What is Teriyaki Anime?
Teriyaki Anime is an anime streaming platform where enthusiasts can enjoy their favorite series anytime, anywhere! Users can sign up, bookmark, rate, leave comments, explore popular titles, genres, and search for a specific anime.

Teriyaki Anime aims to provide a seamless and personalized anime streaming experience for fans worldwide!

> [!IMPORTANT]
> Teriyaki Anime is **NOT** a streaming service. Videos that are shown are provided from Jikanv4 API and are only shown as trailers.
